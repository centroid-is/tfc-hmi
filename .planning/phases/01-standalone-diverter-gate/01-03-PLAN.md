<task type="auto">
  <name>Task 1: ConveyorGate widget with animation and OPC UA binding</name>
  <files>lib/page_creator/assets/conveyor_gate.dart</files>
  <action>
    In `conveyor_gate.dart`, create the `ConveyorGate` widget and update `ConveyorGateConfig.build()`:

    1. **ConveyorGate widget**:
       - `class ConveyorGate extends ConsumerStatefulWidget` with `final ConveyorGateConfig config` constructor param
       - `class _ConveyorGateState extends ConsumerState<ConveyorGate> with SingleTickerProviderStateMixin`

    2. **initState**:
       - Create `_controller = AnimationController(vsync: this, duration: Duration(milliseconds: widget.config.openTimeMs))`
       - Create `_progress = ValueNotifier<double>(0.0)`
       - Add listener: `_controller.addListener(() { _progress.value = Curves.easeOut.transform(_controller.value); })`

    3. **_onStateChanged(bool isOpen)** method:
       - If isOpen: set `_controller.duration = Duration(milliseconds: widget.config.openTimeMs)`, call `_controller.forward()`
       - If !isOpen: set `_controller.duration = Duration(milliseconds: widget.config.closeTimeMs ?? widget.config.openTimeMs)`, call `_controller.reverse()`
       - This ensures every OPC UA state change triggers animation (per user decision: no snap-to-state, even after reconnect)

    4. **build()** method:
       - If `widget.config.stateKey.isEmpty`: render gate with Colors.grey (DATA-06)
       - Otherwise: wrap in StreamBuilder<DynamicValue> using exact pattern from GateStatus.build():
         ```
         ref.watch(stateManProvider.future).asStream().asyncExpand(
           (stateMan) => stateMan.subscribe(widget.config.stateKey).asStream().switchMap((s) => s),
         )
         ```
       - In builder: determine color:
         - `!snapshot.hasData` -> Colors.grey (DATA-06: grey when disconnected)
         - `snapshot.data!.asBool` -> widget.config.openColor (green default)
         - else -> widget.config.closedColor (white default)
       - Call `_onStateChanged(snapshot.hasData && snapshot.data!.asBool)` to trigger animation
       - Return `LayoutRotatedBox(angle: (widget.config.coordinates.angle ?? 0.0) * pi / 180, child: CustomPaint(size: widget.config.size.toSize(MediaQuery.of(context).size), painter: PneumaticDiverterPainter(progress: _progress, stateColor: color, openAngleDegrees: widget.config.openAngleDegrees, side: widget.config.side)))`

    5. **dispose**:
       - `_controller.dispose()` and `_progress.dispose()`

    6. **Update ConveyorGateConfig.build()**:
       - Replace placeholder with: `return ConveyorGate(config: this);`

    IMPORTANT: AnimationController lives in State, NOT inside StreamBuilder.builder (Pitfall 1 from RESEARCH.md). StreamBuilder only calls _onStateChanged and provides color -- it does NOT create animation resources.

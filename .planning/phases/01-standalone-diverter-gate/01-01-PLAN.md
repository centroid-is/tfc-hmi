       - Define `@JsonEnum() enum GateSide { left, right }` with `@JsonKey(unknownEnumValue: GateSide.left)` on config field
       - Define color helpers: `int _colorToJson(Color c) => c.value;` and `Color _colorFromJson(int v) => Color(v);`
       - Create `@JsonSerializable(explicitToJson: true) class ConveyorGateConfig extends BaseAsset` with fields:
         - `GateVariant variant` (default pneumatic)
         - `GateSide side` (default left)
         - `String stateKey` (default empty)
         - `double openAngleDegrees` (default 45.0)
         - `int openTimeMs` (default 800)
         - `int? closeTimeMs` (default null)
         - `@JsonKey(fromJson: _colorFromJson, toJson: _colorToJson) Color openColor` (default Colors.green)

       - Add `toJson() => _$ConveyorGateConfigToJson(this);`
       - Add `static ConveyorGateConfig preview()` factory returning default config with reasonable preview size
       - Stub `build()` method returning a placeholder Container (full widget in Plan 03)
       - Stub `configure()` method returning null (full dialog in Plan 02)
       - Add `part 'conveyor_gate.g.dart';`

    2. Create `lib/page_creator/assets/conveyor_gate_painter.dart`:
       - Create `PneumaticDiverterPainter extends CustomPainter` with constructor params:
         - `required ValueNotifier<double> progress` (0.0=closed, 1.0=open), used as `super(repaint: progress)`
         - `required Color stateColor`
         - `required double openAngleDegrees`
         - `required GateSide side`
       - Define static `_cylinderColor = Color(0xFF9E9E9E)` and `_rodColor = Color(0xFFBDBDBD)`
       - Implement `paint(Canvas canvas, Size size)`:
         - Proportions (Claude's discretion): cylinder body ~15% of height at top, rod ~30% of cylinder height diameter, flap full width and ~8% height thickness
         - Hinge point at left or right edge based on `side`
         - Calculate flap angle: `openAngleDegrees * (pi / 180) * progress.value`
         - Draw cylinder body as rounded rect in metallic grey (always)
         - Draw rod extending from cylinder, length proportional to progress
         - Draw flap using canvas.save/restore with rotation around hinge point, filled with stateColor
         - For `GateSide.right`, mirror the geometry horizontally
       - Implement `shouldRepaint`: compare only `stateColor`, `openAngleDegrees`, `side` (NOT progress -- it's the repaint listenable)

    3. Run `dart run build_runner build --delete-conflicting-outputs` to generate `.g.dart`

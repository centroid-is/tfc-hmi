## Decisions Made

- **AssetLoader returns `List<int>` not `ByteData`:** Allows tests to supply raw bytes without initializing Flutter binding; production code wraps rootBundle.load() in a private method that only runs when assetLoader is null
- **Platform flag injection via constructor:** `platformIsWindows`/`platformIsMacOS` bool params default to `Platform.isX` but can be overridden in tests to exercise platform branches on any OS
- **Header normalization in tests:** `http` package lowercases header keys; tests check both `authorization` and `Authorization` keys to handle case variation across http client versions
- **Package .gitignore added:** Root `.gitignore` excludes `packages/*/pubspec.lock` but not `.flutter-plugins-dependencies`; added per-package `.gitignore` matching `centroid-hmi/.gitignore` pattern

<objective>
Create the `centroidx_upgrader` Dart package with TDD: a `GitHubReleaseStore` (UpgraderStore subclass checking GitHub Releases) and a `ManagerLauncher` (extracts bundled manager binary, strips macOS quarantine, launches as detached process).

Purpose: This package replaces `microsoft_store_upgrader` with a cross-platform GitHub Releases update mechanism. The store tells the `upgrader` package when a new version exists; the launcher handles the handoff to the Go manager binary.

Output: A fully tested pure-Dart package at `packages/centroidx_upgrader/` with two classes and their TDD test suites.
</objective>

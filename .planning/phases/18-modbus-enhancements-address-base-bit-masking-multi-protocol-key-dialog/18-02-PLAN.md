<objective>
Extend KeyMappingEntryDialog in common.dart to show servers from all protocols (OPC UA, Modbus, M2400) instead of only OPC UA.

Purpose: Currently, the page editor's key mapping dialog only allows OPC UA node configuration. Users need to assign Modbus and M2400 keys from the same dialog, especially when building pages that mix protocols. The key_repository.dart already has this multi-protocol pattern -- this brings it to the page editor dialog.

Output: Multi-protocol KeyMappingEntryDialog with protocol-aware server dropdown and dynamic config fields, plus widget tests.

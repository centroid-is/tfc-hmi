<objective>
Add configurable 0-based vs 1-based register addressing per Modbus server.

Purpose: Different PLC vendors use different addressing conventions. Schneider M340/M580 use 1-based (address 40001 = holding register 0), while Siemens/ABB/Beckhoff use 0-based. Without this, users must manually subtract 1 from all addresses when configuring 1-based PLCs.

Output: ModbusConfig.addressBase field, wire-level offset application, Address Base dropdown in server config UI, unit tests.
</objective>

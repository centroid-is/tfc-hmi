<objective>
Add protocol-agnostic bit masking with visual bit grid UI for both Modbus and OPC UA keys.

Purpose: SCADA/HMI operators frequently need to extract individual bits or bit ranges from register values (e.g., bit 3 of a status word = "pump running"). This is standard in tools like KEPServerEX and Ignition. Without it, users must create custom expressions for basic bit-level access.

Output: bitMask/bitShift fields on KeyMappingEntry, masking applied in both Modbus adapter and OPC UA StateMan read/subscribe paths, BitMaskGrid widget, bit mask section in key repository UI.
</objective>

       ```dart
       /// Apply bit mask extraction to a raw value.
       /// Returns the original value unchanged if bitMask is null.
       /// Single-bit mask returns bool, multi-bit returns int.
       static DynamicValue applyBitMask(DynamicValue value, int? bitMask, int? bitShift) {
         if (bitMask == null) return value;
         final raw = value.value;
         if (raw is! num) return value;
         final intValue = raw.toInt();
         final masked = (intValue & bitMask) >>> (bitShift ?? 0);
         final isSingle = bitMask != 0 && (bitMask & (bitMask - 1)) == 0;
         if (isSingle) {
           return DynamicValue(value: masked != 0, typeId: NodeId.boolean);
         }
         return DynamicValue(value: masked, typeId: value.typeId);
       }
       ```

         ```dart
         @override
         Future<void> write(String key, DynamicValue value) async {
           final spec = _specs[key];
           if (spec == null) throw ArgumentError('Unknown Modbus key: $key');
           if (spec.bitMask != null) {
             // Read-modify-write
             final current = wrapper.read(key);
             final currentInt = (current is num) ? current.toInt() : 0;
             final shift = spec.bitShift ?? 0;
             final isSingle = spec.bitMask! != 0 && (spec.bitMask! & (spec.bitMask! - 1)) == 0;
             int newValue;
             if (isSingle) {
               final boolVal = value.value == true || value.value == 1;
               if (boolVal) {
                 newValue = currentInt | spec.bitMask!;
               } else {
                 newValue = currentInt & ~spec.bitMask!;
               }
             } else {
               final writeInt = (value.value is num) ? (value.value as num).toInt() : 0;
               newValue = (currentInt & ~spec.bitMask!) | ((writeInt << shift) & spec.bitMask!);
             }
             await wrapper.write(spec, newValue);
           } else {
             await wrapper.write(spec, value.value);
           }
         }
         ```

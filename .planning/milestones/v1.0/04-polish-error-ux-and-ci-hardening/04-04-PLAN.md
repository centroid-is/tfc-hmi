# Plan 04-04 — Simulate Motion Toggle

## Objective

Add a "Simulate motion" toggle to the elevator config dialog. When ON,
the elevator's platform animates smoothly between 0% and 100% on a fixed
cycle, ignoring the live PLC state-key stream. When OFF, behaviour
reverts to the existing PLC-driven pipeline.

Reference pattern: `lib/page_creator/assets/conveyor.dart` already
implements an equivalent `simulateBatches` toggle.
